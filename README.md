# Data Analysis Bike Sharing

```bash
# Buat direktori proyek dan pindah ke dalamnya
mkdir Data_Analisis_Bike_Sharing

# Install pipenv dan masuk ke virtual environment
pipenv install
pipenv shell

# Install semua dependensi dari requirements.txt
